# Zendesk Tickets 
This web app is designed to access my Zendesk tickets through the API.

## Tech Used
- JavaScript
- Node.js
- Express
- dotenv
- request


## Install, Setup & Run

```
git clone https://github.com/LWRGitHub/zendesk-tickets.git
cd zendesk-tickets
npm install
mv .env.example .env
```
***Make sure to put your zendesk email & pasword in the .env file***
